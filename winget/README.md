# WinGet Package for Quantcept

## Install

```powershell
winget install FinceptCorporation.Quantcept
```

## Publishing

To publish to WinGet, submit a pull request to [microsoft/winget-pkgs](https://github.com/microsoft/winget-pkgs) with the manifest file placed at:

```
manifests/f/FinceptCorporation/Quantcept/0.1.0/FinceptCorporation.Quantcept.yaml
```

Use the [wingetcreate](https://github.com/microsoft/winget-create) tool to automate submissions:

```powershell
wingetcreate submit FinceptCorporation.Quantcept.yaml
```
