# Homebrew Tap for Quantcept

## Setup

This formula is distributed via the [Fincept Homebrew tap](https://github.com/Fincept-Corporation/homebrew-quantcept).

### Install

```bash
brew tap Fincept-Corporation/quantcept
brew install quantcept
```

### Update

```bash
brew upgrade quantcept
```

### Uninstall

```bash
brew uninstall quantcept
```

## Tap Repository

To publish this formula, create a repo named `homebrew-quantcept` under the Fincept-Corporation GitHub org and place `quantcept.rb` in `Formula/quantcept.rb`.
