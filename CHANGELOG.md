# Changelog

All notable changes to Quantcept will be documented in this file.

## [0.1.0] - 2026-05-26

### Added

- Initial release
- Interactive TUI with React/Ink
- Tool system: file read/write/edit, glob, grep, shell, calculator, PDF parse, CSV analyze, web fetch, web search
- Permission system with allow/deny/ask rules
- Slash commands: /help, /analyze, /model, /sessions, /export, /cost, /tasks, /doctor, /thinking, /clear, /exit
- Token usage tracking
- Session persistence and resume
- MCP server support
- Plugin system with marketplace
- Skills system (bundled + filesystem + plugin)
- Hook system (pre/post tool execution)
- Customizable keybindings
- Thinking mode display
- Conversation export (markdown, JSON, text)
- Financial calculator (NPV, IRR, compound interest, % change)
